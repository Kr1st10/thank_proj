# Первый дедлайн backend

В этой папке собраны артефакты первого дедлайна backend-направления для проекта «Система совместной работы».

Состав:
- `TZ.md` — техническое задание с описанием продукта, ролей, сценариев и границ MVP.
- `API.md` — спроектированный REST API, контрактные модели и примеры запросов/ответов.
- `er-diagram.drawio` — ER-диаграмма базы данных в формате draw.io.

Этот пакет закрывает документальную часть первого дедлайна. Реализация backend на C#, инфраструктура, БД и запускаемый сервис будут следующей итерацией.
